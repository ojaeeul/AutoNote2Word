☁️ Streamlit Cloud 전 세계 어디서든 접속 배포 가이드
===========================================================

클라우드 배포를 하면 갤럭시탭, 아이폰, 회사 PC 등 어디서든 
고정된 인터넷 주소(예: https://autonote-to-word.streamlit.app)로 접속할 수 있습니다.

제가 배포에 필요한 모든 설정 파일(packages.txt, config.toml 등)을 
준비해 두었으니, 아래 단계만 따라하시면 5분 만에 끝납니다!

1. GitHub에 코드 올리기
   - [GitHub.com](https://github.com/)에 로그인합니다.
   - 새 저장소(New Repository)를 만듭니다 (이름 예: AutoNote2Word).
   - 현재 폴더의 모든 파일(app.py, requirements.txt, .streamlit 폴더 등)을 마우스로 끌어서 GitHub에 업로드합니다.

2. Streamlit Cloud 연결하기
   - [Streamlit Cloud](https://share.streamlit.io/)에 접속하여 GitHub 계정으로 로그인합니다.
   - [Create app] 버튼을 누릅니다.
   - 방금 만든 GitHub 저장소와 `app.py` 파일을 선택합니다.
   - [Deploy!] 버튼을 누릅니다.

3. API 키 설정 (선택 사항)
   - 앱이 배포되면 화면 오른쪽 하단의 [Settings] -> [Secrets] 메뉴에 아래 내용을 입력하면 매번 API 키를 넣을 필요가 없어 편리합니다.
     GEMINI_API_KEY = "내_제미나이_키"
     OPENAI_API_KEY = "내_오픈AI_키"

4. 결과
   - 이제 나만의 전용 URL이 생깁니다! 
   - 이 주소를 갤럭시탭 S10의 브라우저 즐겨찾기에 등록해 두고 언제 어디서든 사용하세요.

-----------------------------------------------------------
※ 주의: GitHub 저장소를 'Public(공개)'으로 만들 경우, API 키가 노출될 수 있으니 반드시 'Private(비공개)'으로 만들거나, 코드 내의 키를 지우고 배포해 주세요.
