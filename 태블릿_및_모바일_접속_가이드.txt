📱 갤럭시탭 S10 및 태블릿/모바일 접속 가이드
===========================================================

본 프로그램(Streamlit)은 웹 기반 앱이므로, 메인 PC에서 실행 중이라면
동일한 Wi-Fi를 사용하는 갤럭시탭이나 스마트폰에서도 접속이 가능합니다.

1. 메인 PC(윈도우)에서 앱 실행
   - [ run_windows.bat ] 파일을 실행합니다.
   - 실행 창에 표시되는 "Your PC's Network IP" 주소를 확인합니다.
     (예: 192.168.0.15)

2. 갤럭시탭 S10에서 접속하기
   - 갤럭시탭이 PC와 동일한 Wi-Fi에 연결되어 있는지 확인합니다.
   - 갤럭시탭의 브라우저(Chrome, 삼성 인터넷 등)를 엽니다.
   - 주소창에 아래와 같이 입력합니다 (IP 주소는 PC에 표시된 것을 사용하세요).
     👉 http://[확인된_IP_주소]:8501
     (예: http://192.168.0.15:8501)

3. 특징 및 장점
   - 펜 입력 활용: 갤럭시탭 S10의 S펜을 사용하여 수식이나 내용을 더 편하게 검토할 수 있습니다.
   - 무선 사용: PC는 켜두고 거실이나 침대에서 태블릿으로 작업을 이어갈 수 있습니다.

4. 접속이 안 될 경우 확인 사항
   - Wi-Fi 확인: 두 기기가 같은 Wi-Fi 공유기에 연결되어야 합니다.
   - 윈도우 방화벽: 처음 실행 시 "Windows 보안 경고" 창이 뜨면 [액세스 허용]을 눌러주어야 합니다.
   - 방화벽 설정: 계속 접속이 안 되면 윈도우 설정 -> [업데이트 및 보안] -> [Windows 보안] -> [방화벽 및 네트워크 보호]에서 'Python' 또는 'Streamlit'에 대한 허용이 되어있는지 확인하세요.
