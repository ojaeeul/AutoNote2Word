🚀 AutoNote2Word Pro - 윈도우 환경 실행 가이드
===========================================================

본 폴더를 회사나 다른 윈도우 PC로 옮겨서 사용하실 때 아래 절차를 따라주세요.

1. 준비 사항
   - PC에 파이썬(Python 3.9 이상 권장)이 설치되어 있어야 합니다.
   - 인터넷 연결이 필요합니다 (API 호출 및 라이브러리 설치용).

2. 실행 방법
   - 폴더 내의 [ run_windows.bat ] 파일을 마우스 오른쪽 버튼으로 클릭하거나 더블 클릭하여 실행합니다.
   - 처음 실행 시 필요한 라이브러리들을 자동으로 설치하므로 시간이 조금 걸릴 수 있습니다.
   - 설치가 완료되면 자동으로 브라우저가 열리며 앱이 실행됩니다.

3. 무중단 분석 이어하기
   - [ workspace_state.json ] 파일과 [ temp_uploads ] 폴더를 함께 복사해 가시면,
     집에서 분석하던 진행 상황(페이지 위치, 이미 분석된 결과 등)을 회사에서 그대로 이어서 할 수 있습니다.

4. 주요 기능 설정
   - 앱이 실행되면 사이드바(Sidebar)에서 Gemini API 키와 OpenAI API 키를 입력해 주세요.
   - [🚀 무중단 초정밀 심층 분석 시작] 버튼을 누르면 80페이지 이상의 대형 문서도 누락 없이 분석합니다.

5. 문제 해결 (Troubleshooting)
   - Word 파일 다운로드 시 오류가 발생할 경우: 
     https://pandoc.org/installing.html 에서 Windows용 Pandoc을 설치해 주세요.
   - 앱이 실행되지 않을 경우: 
     명령 프롬프트(CMD)를 열고 해당 폴더로 이동하여 `pip install -r requirements.txt`를 직접 입력해 보세요.
